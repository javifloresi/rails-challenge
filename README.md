# README

## INSTALLATION

### Development
1. Install docker & docker-compose
2. docker-compose up
3. Go to http://localhost

### Production
1. docker build -f Dockerfile.prod -t javifloresi/rails-challenge .
2. docker run --env-file env -d -p 80:80 javifloresi/rails-challenge
3. Go to http://localhost
